# 🎮 PS5 Game Launcher Clone

A slick, animated PS5-style game launcher built with React and Framer Motion for smooth, console-like transitions. This project is all about experimenting with playful UI effects using SVG magic and clever CSS tricks to bring that next-gen vibe to the web.

# ✨ Features
Smooth page transitions with Framer Motion

Animated SVG and custom UI effects

Inspired by the PS5 game menu design

# 🛠️ Tech Stack
- React

- Framer Motion

- CSS / TailwindCSS

- SVG graphics & animations
